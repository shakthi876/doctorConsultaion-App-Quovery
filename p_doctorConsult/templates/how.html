How to use

Welcome to Doctor Consultation App !

User Guidlines :

  1. Register the user details
  2. Enter your Hospital-name with unique email-id
    Each Registered Hospital must have three users namely Attendant, Doctor, Medical-Shop.
  3. Enter your Attendent, Doctor, Medical-shop Password. All three passwords should be different
  4. Login with email-id and respected passwords based on users(Attendant/Doctor/Medical-Shop)

Note:

There are there users having different password :
  • Attendant - Add the patients details(Name, Age, Address, Contact-number),
  • Doctor - Diagonsed that patient and add problems with tablet names. create the prescription,
  • Medical-Shop - Check the tablets availablity by seeing doctor's prescription and generate the bill.
